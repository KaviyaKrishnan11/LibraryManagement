This is a simple Library Management System developed using Java Servlets, JDBC, HTML, and Apache Tomcat.
The project allows users to add book details and view the added book information through a web interface.
All book data is processed using Servlets and stored/retrieved from a database using JDBC.
This project is mainly created for learning and understanding Java Web Applications, Servlet lifecycle, and database connectivity using JDBC.
